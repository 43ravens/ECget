**********************************************************
ECget -- Get Environment Canada Weather & Hydrometric Data
**********************************************************

ECget is a tool to get observation data from Environment Canada weather and hydrometric web services and sites and store it in project-specific file formats.
It is written in Python and is designed to be easily extensible for use in various projects and applications.
